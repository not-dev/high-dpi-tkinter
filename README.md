# High-DPI-Tkinter

Sharpen blurry Tkinter scaling on Windows 10 high DPI displays

## Installation

Install using pip

```sh
pip install hdpitkinter
```

## Usage

```python
from hdpitkinter import HdpiTk

root = HdpiTk()
root.mainloop()
```
