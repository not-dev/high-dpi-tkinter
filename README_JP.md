# High-DPI-Tkinter

Windows10の高DPIディスプレイでぼやけるTkinterを鮮明に

## インストール

pipでインストール

```sh
pip install hdpitkinter
```

## 使用方法

```python
from hdpitkinter import HdpiTk

root = HdpiTk()
root.mainloop()
```
